class three.MF.V3.Descriptors.BoundingBox.BoundingBox(center=None, size=None, rotation=None, transform=None)¶

Bases: object

BoundingBox descriptor.

rotation¶

The 4x4 matrix that transforms the canonical cube with corners [±1, ±1, ±1] to the bounding box in world coordinates. The transform can be used as the model matrix for rendering the bounding box with an OpenGL shader.

size¶

The 3x3 rotation matrix of the bounding box. The first, second and third column vectors are the x, y and z axes of the bounding box.

class three.MF.V3.Descriptors.Calibration.CaptureTarget(camera, quads=None)¶

Bases: object

Calibration capture target.

The camera calibration capture targets are used to draw quad overlays on the video stream to guide a user as to where to position the calibration card for each capture during camera calibration.

camera¶

The target quad for each camera. This is a set of 16 numbers defining the quad coordinates on the left and right camera. The first 4 pairs of numbers define the quad on the left camera. The last 4 pairs of numbers define the quad on the right camera.
